## Frequently Asked Questions

### My server keeps kicking me or I cannot join my server

There is a known issue in v1.0.8-beta, the server files have since been taken down due to discovering a bug in one of the mods which has caused this issue.  
We recommend updating to v1.0.9-beta for modpack and server, please backup your worlds first before doing so!

If you are having trouble, please first check your server crash reports to see what is causing the problem.
